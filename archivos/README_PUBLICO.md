## Archivos de carga y privacidad

Este directorio conserva solo insumos seguros para un repositorio publico.

- `inserciones_usuarios.sql`: dataset demo sin datos reales.
- `inserciones_mantenimientos.sql`: dataset demo de mantenimientos.
- `modelo_mantenimientos.sql`: estructura base de la base de datos.

Los archivos fuente originales de operacion, como cronogramas internos o listados reales de usuarios y equipos, se excluyen del repositorio por privacidad.
